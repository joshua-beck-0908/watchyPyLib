\doxysubsection{Detailed Description}
A class for syncing the \mbox{\hyperlink{classwatchylib_1_1RTC}{RTC}} with an \mbox{\hyperlink{classwatchylib_1_1NTP}{NTP}} server. 

This sync should be done every day if possible, depending on the \mbox{\hyperlink{classwatchylib_1_1RTC}{RTC}} accuracy. A DS3231 may only need once a week.
