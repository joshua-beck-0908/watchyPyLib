\doxysubsection{Detailed Description}
e\+Paper display driver. 

Provides functions for drawing to the display. The display is a 200x200 black and white EPD display with a SSD1681 driver. A basic driver is provided in the firmware to draw to the display. That at least lets us see the terminal output. But it\textquotesingle{}s slow and doesn\textquotesingle{}t support many features. This replacement driver allows a lot more control.
