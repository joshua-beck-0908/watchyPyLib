Detailed Description

Defines the four buttons on the side of the watch.

Provides convenience functions for accessing the buttons. See the Button class for more information.

Available buttons are: up, down, cancel, confirm, menu, back, any You can access them like this: watchy.button.up.pressed watchy.button.down.wait() any is a meta button that can be used to wait for any button to be pressed. i.e. watchy.button.any.wait()
