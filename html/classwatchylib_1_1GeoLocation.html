Detailed Description

A class for getting the current location from an IP address.

This is done by querying an IP geolocation API. The API used is ip-api.com. This is a great free service that can be used (non-commerically) without an API key. The resulting data is used for time, weather, and date formatting.

Attempts to refresh the GeoLocation data.

The geolocation refresh may fail if the internet is not available. Refreshing may be needed if the user travels to a new location. e.g. If the user travels to a different town or suburb, the temperature and weather may be different.
